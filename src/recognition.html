Recognition

This app bundles artwork and icons from the projects below. We are grateful to their authors and maintainers.

🎮

Platform logos

Platform logos on NFC card labels use bundled artwork from the Carbon EmulationStation theme (RetroPie). The platform selector and collection list use RetroArch XMB themes from the libretro assets repository. The default UI theme is Dot Art; other themes are available in Global Settings. Monochrome, Pixel, and Automatic UI icons are inverted in light mode so they stay visible on pale backgrounds.

  • RetroPie/es-theme-carbon — bundled Carbon system SVGs on printed card headers
  • libretro/retroarch-assets — XMB system icons (xmb/*/png/*.png) for the in-app platform selector
🖼️

Game artwork

Box art, title screens, and in-game snapshots are sourced from the libretro-thumbnails project on GitHub and loaded at runtime via raw.githubusercontent.com.

🖨️

3D-printed sticker applicator

The free TapTo Sticker Applicator jig on the Supplies page was designed by EntirelyTom. His Printables profile has the model and other maker designs.

  • TapTo Sticker Applicator on Printables
💡

Inspiration

This project was influenced by Wizzo and the MiSTeR FPGA community. Wizzo created TapTo (now Zaparoo), the open-source NFC loading system that lets you tap physical cards to launch games on MiSTer and other platforms — the workflow this designer supports.

🔤

Fonts & UI

The Carbon theme also ships Cabin Bold and other UI assets we do not currently bundle; only the platform system SVGs listed above are used here.
